## **Project Title:**

**LAN Chat Application Using Python Sockets**

---

## **Project Overview:**

The LAN Chat Application is a lightweight networking project that enables real-time text communication between two or more devices connected on the same Local Area Network (LAN). This application demonstrates practical implementation of **socket programming**, **multithreading**, and **network communication protocols** in Python, making it an excellent example of applied networking concepts.

---

## **Objectives:**

1. Develop a reliable and interactive chat system for devices on a LAN.
2. Implement server-client architecture using Python sockets.
3. Ensure real-time bidirectional communication with minimal latency.
4. Provide a professional and demonstrable interface suitable for presentations or live demos.

---

## **Key Features:**

* **Real-Time Communication:** Messages are sent and received instantly between connected devices.
* **Multi-Threading:** Separate threads handle sending and receiving messages simultaneously, ensuring smooth operation.
* **Single-Terminal Option:** For easy demonstrations, the application can run both server and client in the same terminal, avoiding network configuration issues.
* **Cross-Platform Compatibility:** Can be run on Windows, Mac, or Linux with Python installed.
* **Safe and Lightweight:** Requires minimal system resources, making it ideal for educational and demonstration purposes.

---

## **Technical Details:**

1. **Programming Language:** Python 3.x
2. **Key Libraries:**

   * `socket` → Implements TCP/IP communication.
   * `threading` → Handles simultaneous sending and receiving of messages.
   * `time` (optional) → Used for simulating responses in demo mode.
3. **Architecture:**

   * **Server:** Listens on a specific port, waits for client connections, and handles message reception and transmission.
   * **Client:** Connects to the server using IP and port, and enables sending and receiving messages in real-time.
4. **Network Protocol:** TCP (Transmission Control Protocol) ensures reliable message delivery.
